OVI DISEÑO DE SITIOS WEB

Con la construccion de este sitio web, se pretende dar un vistazo de los temas iniciales de diseño de sitios web, consideramos que toda persona que desee subirse al barco del diseño web, debe conocer acerca de Git y GitHub, asi como la creacion de repositorios y por supuesto saber que papel juega HTML5 y CSS3 En cada unidad encontrara unas lecturas que lo adentraran al tema escogido, tambien encuentra una seccion multimedia con tuturiales y conceptos de los temas, para terminar se encontrara una evaluacion para que evaluie su proceso de aprendizaje. Este sitio fue elaborado por estudiantes del grupo 301122_45, del curso Diseños de sitios web.


OBJETIVO GENERAL:

Ilustrar teóricamente a los usuarios visitantes de la página, sobre las generalidades correspondientes al diseño de sitios web como lo son los sistemas de edición de controles, estructuras, lenguajes de marca en (HTML5 y CSS3), entre otros aspectos.

OBJETIVO ESPECÍFICO 1:

Conocer todo lo relacionado con los sistemas de control de versiones como GIT o GITHUB, para compartir y publicar archivos en formato HTML5 y CSS3, a través de linea de comando o por medio de la consola gráfica de GitHub Desktop..

OBJETIVO ESPECÍFICO 2:

Brindar información complementaria relacionada con el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3.

OBJETIVO ESPECÍFICO 3:

Ofrecer accesibilidad a recursos multimedia y ejemplos de diseño de sitios web, basados en uso de GITHUB y lenguajes HTML5 y CSS3.
